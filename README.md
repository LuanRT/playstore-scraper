# Play Store Scraper

A simple and fast way to get search results and more from Google Play Store. However, keep in mind that this module may break without any warning if updates are made to the Google Play Store website.

## Installation

```bash
npm install playstore-scraper
```

## Usage

```js
const playstore = require("playstore-scraper");

// A simple search:

playstore
  .search("TWDG")
  .then((res) => {
    console.log(res);
  })
  .catch((err) => console.log(err));
```

#### If you need to get more info about a certain item:

```js
playstore
  .search("TWDG")
  .then((res) => {
    playstore
      .getExtendedInfo(res.results[0].link)
      .then((res) => {
        console.log(res);
      })
      .catch((err) => console.log(err));
  })
  .catch((err) => console.log(err));

// Or:

playstore
  .getExtendedInfoById("com.telltalegames.walkingdead200")
  .then((res) => {
    console.log(res);
  })
  .catch((err) => console.log(err));
```
